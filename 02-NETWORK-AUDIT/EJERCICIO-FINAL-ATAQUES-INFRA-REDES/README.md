# Ejercicio Final - Ataques a Infraestructuras de Sistemas y Redes

## Objetivo
Auditar una red doméstica real para identificar equipos, enumerar servicios, evaluar vulnerabilidades y realizar una prueba de MITM para análisis de tráfico.

## Actividades realizadas
- Descubrimiento de hosts en red local
- Identificación de dispositivos por MAC / fabricante
- Selección de un objetivo
- Escaneo de puertos y servicios
- Enumeración de versiones
- Evaluación con herramientas automáticas (vulnerabilidades)
- Triaje por criticidad (CVSS)
- MITM / sniffing de tráfico y análisis en Wireshark
