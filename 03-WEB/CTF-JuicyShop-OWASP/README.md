# CTF Grupal - OWASP Juice Shop (Pentest Web)

## Objetivo
Realizar un pentest web sobre OWASP Juice Shop y documentar los hallazgos siguiendo una estructura tipo OWASP Testing Guide (OTG).

## Entorno
- Kali Linux
- OWASP Juice Shop (localhost:3000)
- Herramientas: Burp Suite + análisis manual

## Hallazgos realizados (ejemplos)
- XSS (DOM / Reflected)
- SQL Injection (login bypass)
- Broken Access Control (Admin Section)
- Parameter tampering (Zero stars)
- Exposición de información (Confidential documents / metrics)

## Mi contribución (IMPORTANTE)
- Participación en la documentación y evidencias (capturas + explicación)
- Identificación y explicación de vulnerabilidades OWASP
- Recomendaciones y mitigación
